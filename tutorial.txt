Salam, sedikit explanation mengenai fungsi bot ini dan macam mana untuk run.

Mula2 dapatkan api_key dan api_secret daripada Luno
Boleh pergi dekat Profile>Security>API keys>Create API key
Copy dan paste key dan secret dalam note/word

Install Python launcer dari windows store atau website https://www.python.org/downloads/

Buka api.txt dan set bot mengikut strategi masing2
Dekat bahagian api_key dan api_secret, copy dan paste key dan secret yang diberi oleh Luno di dalam kurungan "xxxxx"

Set peratus profit yang ingin diambil, sebagai contoh 0.01 = 1%, untuk 5 percent set kepada 0.05

Pilih coin untuk trade, tukar di bahagian market_pair dan setkan saiz trade di bahagian trade_quantity. Secara default BTC (XBTMYR), ETH dan XRP telah diset kepada jumlah trade terendah yang dibenarkan oleh Luno.

Untuk run bot, right click Luno.py>Open with>Python launcer

Limitasi API Luno
- hanya 300 call per minute
- hanya boleh detect 100 order terkini
Oleh sebab limitasi diatas, recommended untuk hanya run 3-5 coin serentak dan restart bot setiap hari. 

Bug yang ada:
order lama tidak auto cancel>cancel secara manual di order list

Sekiranya bot problem boleh tutup bot dengan tekan Control + C dan run balik bot
